# Experience Cards Placeholders

Placeholder actual: `/public/placeholders/experience-card-default.svg`

Opciones tentativas de imagen final:
- Capturas de dashboards o productos donde participaste.
- Foto del equipo o entorno de trabajo ligada al rol.
- Mockup de arquitectura o flujo técnico simplificado.
- Imagen abstracta consistente por cada experiencia con color coding.

Especificaciones recomendadas:
- Ratio: `16:10`.
- Resolución mínima: `1200x760`.
- Estilo: consistente entre tarjetas, sin saturación visual.
- Formato sugerido: `webp`.
